# nclean
A utility to clean filenames

```
USAGE:
    nclean.exe [OPTIONS] <PATTERN>

ARGS:
    <PATTERN>

OPTIONS:
    -d, --directory <DIRECTORY>
    -f, --force
    -h, --help                     Print help information
    -r, --replace <REPLACE>
    -t, --no-trim
    -V, --version                  Print version information
```
